代码及数据文件夹内下含两个文件夹 分别是 “confusion_matrix” 和 “model_train_save_apply”。
“confusion_matrix”内为5个模型的混淆矩阵 对应论文中的table 2-7。
“model_train_save_apply”分别是4个代码文件笔记本、“save”、“logfile”文件夹
后缀为.ipynb的四个代码笔记本中编写的语言环境是python3.5 其中：
“MLP-By Pixel and tensorflow.ipynb”和“CNN-By Block and tensorflow.ipynb”两个代码主要功能为数据的读取、数据集的制作、tensorflow下模型的搭建、训练存储以及tensorboard的可视化。
“Apply  MLP model.ipynb”和“Apply  CNN model.ipynb”中代码主要功能为模型的调用以及分类图的输出。
“logfile”为tensorboard的可视化文件
“save”为模型的训练完成后参数的存储文件(文件夹内只包含MLP和CNN-9的文件)。
